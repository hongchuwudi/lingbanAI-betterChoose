## 老人端 vs 子女端 菜单设计

---

### 老人端菜单（5个）

| 菜单 | 功能 | 二级菜单/内容 |
|------|------|--------------|
| **首页** | 健康资讯聚合页 | 推荐、推文、健康提醒等（可做二级点击菜单，或一页滚动展示） |
| **家人** | 查看子女信息 | 子女列表、子女详情、联系子女 |
| **灵伴** | AI 智能对话 | 与 Agent 聊天、健康咨询、情绪陪伴 |
| **消息** | 通知消息中心 | 系统通知、子女消息、健康提醒 |
| **我的** | 个人中心 | 个人信息、健康档案、设置、退出登录 |

---

### 子女端菜单（5个）

| 菜单 | 功能 | 二级菜单/内容 |
|------|------|--------------|
| **首页** | 父母健康概览 | 健康数据看板、异常预警、健康报告 |
| **家人** | 查看老人信息 | 老人列表、老人详情、健康数据 |
| **灵伴** | AI 智能对话 | 与 Agent 聊天、健康建议、照护咨询 |
| **消息** | 通知消息中心 | 发送给老人的通知、系统消息、预警提醒 |
| **我的** | 个人中心 | 个人信息、守护设置、退出登录 |

---

## 功能对比表

| 菜单 | 老人端 | 子女端 |
|------|--------|--------|
| **首页** | 健康资讯（推荐/推文/提醒） | 父母健康看板（数据/预警） |
| **家人** | 查看子女 | 查看老人 |
| **灵伴** | AI 对话（陪伴/咨询） | AI 对话（照护建议） |
| **消息** | 接收消息 | 发送消息给老人 |
| **我的** | 个人中心 | 个人中心 + 守护设置 |

## 架构建议

```
lib/
├── screens/
│   ├── elder/                    # 老人端页面
│   │   ├── elder_home_screen.dart
│   │   ├── elder_family_screen.dart
│   │   ├── elder_lingban_screen.dart
│   │   ├── elder_message_screen.dart
│   │   └── elder_profile_screen.dart
│   └── child/                    # 子女端页面
│       ├── child_home_screen.dart
│       ├── child_family_screen.dart
│       ├── child_lingban_screen.dart
│       ├── child_message_screen.dart
│       └── child_profile_screen.dart
├── widgets/
│   └── bottom_nav_bar.dart       # 底部导航栏（根据角色动态配置）
└── providers/
    └── auth_store.dart           # 根据 user.isElder/isChild 决定菜单
```

## 动态菜单配置

```dart
// 根据角色返回不同菜单
List<NavItem> getNavItems(User user) {
  if (user.isElder) {
    return [
      NavItem(label: '首页', icon: LucideIcons.home, screen: ElderHomeScreen()),
      NavItem(label: '家人', icon: LucideIcons.users, screen: ElderFamilyScreen()),
      NavItem(label: '灵伴', icon: LucideIcons.messageCircle, screen: LingbanScreen()),
      NavItem(label: '消息', icon: LucideIcons.bell, screen: ElderMessageScreen()),
      NavItem(label: '我的', icon: LucideIcons.user, screen: ElderProfileScreen()),
    ];
  } else {
    return [
      NavItem(label: '首页', icon: LucideIcons.home, screen: ChildHomeScreen()),
      NavItem(label: '家人', icon: LucideIcons.users, screen: ChildFamilyScreen()),
      NavItem(label: '灵伴', icon: LucideIcons.messageCircle, screen: LingbanScreen()),
      NavItem(label: '消息', icon: LucideIcons.bell, screen: ChildMessageScreen()),
      NavItem(label: '我的', icon: LucideIcons.user, screen: ChildProfileScreen()),
    ];
  }
}
```

---

这样老人和子女看到的菜单不同，但灵伴（AI 对话）可以共用同一个页面。
