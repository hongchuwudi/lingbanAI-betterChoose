## 给 Trae 的提示词

```
请帮我用 Flutter 实现一个"我的"页面，类似闲鱼、拼多多 App 的个人中心风格。

## 一、页面布局（从上到下）

### 部分1：用户信息卡片（白色卡片，圆角）
- 左侧：圆形头像（60px），点击可上传头像
- 中间：昵称 + 手机号/账号
- 右侧：两个图标按钮
  - 编辑按钮（LucideIcons.pencil）：编辑个人资料
  - 设置按钮（LucideIcons.settings）：进入设置页面
- 底部可选：会员标识（VIP 显示金色徽章）

### 部分2：我的关系人（家人列表）
- 标题："我的家人" + "查看更多"箭头（可点击跳转）
- 横向滚动列表，显示家人头像、昵称、关系
- 每个卡片包含：圆形头像、昵称、关系标签
- 添加家人按钮（灰色圆形，带加号）
- 数据先写死，待后端对接

### 部分3：操作按钮区域
- 退出登录按钮（红色边框，圆角）
- 切换身份按钮（灰色边框，圆角）—— 老人端显示"切换为子女"，子女端显示"切换为老人"

## 二、UI 设计要求

1. 背景色浅灰色（#F5F5F5）
2. 用户信息卡片白色背景，圆角 16px，有轻微阴影
3. 头像圆形，带编辑图标浮层（点击上传）
4. 家人列表横向滚动，卡片间距 12px
5. 按钮宽度占满，高度 48px，圆角 12px，间距 12px
6. 使用 lucide_icons 图标库
7. 适配亮色/暗色主题

## 三、功能要求

1. 点击头像：调用图片选择器，上传头像（先实现 UI，后端接口待对接）
2. 点击编辑按钮：跳转到编辑资料页面（先占位）
3. 点击设置按钮：跳转到设置页面（先占位）
4. 点击家人卡片：跳转到家人详情（先占位）
5. 点击添加家人：跳转到添加家人页面（先占位）
6. 点击退出登录：弹窗确认，确认后清除 token，跳转到登录页
7. 点击切换身份：弹窗确认，切换角色后重新加载菜单

## 四、代码要求

1. 使用 StatefulWidget
2. 从 AuthStore 获取当前用户信息
3. 头像上传使用 image_picker 包（需添加依赖）
4. 使用 lucide_icons
5. 代码注释清晰

## 五、依赖项

在 pubspec.yaml 中添加：
- image_picker: ^1.0.0
- lucide_icons: ^0.257.0

## 六、参考设计

类似闲鱼个人中心：
- 顶部白色卡片用户信息
- 中间横向滚动功能入口
- 底部操作按钮
```
