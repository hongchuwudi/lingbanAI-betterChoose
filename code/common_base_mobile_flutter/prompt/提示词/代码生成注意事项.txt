写代码需要注意的事项:
1.代码需要写注释,注释要写清楚代码语法和业务功能
2.创建文件需要在README中更新内容
3.如果有通用组件的情况下 先封装组件再使用它开发业务功能
4.写代码一定要参考我其他代码的风格,不准随意创作,要注意规范
5.写代码一定要使用简单实用的方案,不要为了追求技术而写出难度很大的代码
6.写代码一定要选择最佳方案 避免意外的bug

springboot:
1.通过上下文获取用户ID
