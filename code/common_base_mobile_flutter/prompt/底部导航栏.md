## Flutter 底部导航栏组件 - 开闭原则设计

---

## 提示词（发送给 TraeAI）

```
请帮我用 Flutter 实现一个底部导航栏组件，要求如下：

## 一、功能需求

1. 底部导航栏包含 5 个模块：首页、健康、消息、发现、我的
2. 使用 lucide_icons 图标库
3. 支持点击切换页面，页面切换有动画效果
4. 支持当前选中项高亮显示（图标和文字变色）
5. 遵循开闭原则，易于扩展新的导航项

## 二、架构要求

1. 将底部导航栏封装成独立组件 `BottomNavBar`
2. 导航项配置与 UI 分离，使用 `NavItem` 数据模型
3. 页面内容使用 `PageView` + `IndexedStack` 组合，保证页面状态保持
4. 页面列表可配置，支持动态增删导航项

## 三、代码结构

```
lib/
├── widgets/
│   └── bottom_nav_bar.dart          # 底部导航栏组件
├── screens/
│   ├── home_screen.dart             # 首页
│   ├── health_screen.dart           # 健康页
│   ├── message_screen.dart          # 消息页
│   ├── discover_screen.dart         # 发现页
│   └── profile_screen.dart          # 我的页
└── main.dart                         # 入口文件
```

## 四、UI 设计

- 导航栏背景色跟随主题（亮色/暗色自动适配）
- 选中项使用主题主色
- 未选中项使用灰色
- 图标大小 24px，文字大小 12px
- 点击时有水波纹效果

## 五、开闭原则体现

- 新增导航项只需在配置列表中添加，无需修改组件内部代码
- 导航项配置支持自定义图标、文字、页面组件
- 支持 badge 小红点功能（可选）

## 六、技术栈

- Flutter 3.x
- lucide_icons
- provider（状态管理）
```

### 1. `lib/widgets/bottom_nav_bar.dart`（底部导航栏组件）

```dart
import 'package:flutter/material.dart';
import 'package:lucide_icons/lucide_icons.dart';

/// 导航项数据模型
class NavItem {
  final String label;
  final IconData icon;
  final IconData activeIcon;
  final Widget screen;
  final int? badgeCount;

  const NavItem({
    required this.label,
    required this.icon,
    this.activeIcon,
    required this.screen,
    this.badgeCount,
  });

  IconData get activeIconData => activeIcon ?? icon;
}

/// 底部导航栏组件
/// 遵循开闭原则，新增导航项只需修改配置列表
class BottomNavBar extends StatefulWidget {
  final List<NavItem> items;
  final Color? backgroundColor;
  final Color? selectedColor;
  final Color? unselectedColor;
  final double iconSize;
  final double fontSize;

  const BottomNavBar({
    super.key,
    required this.items,
    this.backgroundColor,
    this.selectedColor,
    this.unselectedColor,
    this.iconSize = 24,
    this.fontSize = 12,
  });

  @override
  State<BottomNavBar> createState() => _BottomNavBarState();
}

class _BottomNavBarState extends State<BottomNavBar> {
  late int _currentIndex;
  late PageController _pageController;
  late List<Widget> _screens;

  @override
  void initState() {
    super.initState();
    _currentIndex = 0;
    _pageController = PageController(initialPage: 0);
    _screens = widget.items.map((item) => item.screen).toList();
  }

  @override
  void dispose() {
    _pageController.dispose();
    super.dispose();
  }

  void _onTap(int index) {
    if (_currentIndex == index) return;

    setState(() {
      _currentIndex = index;
    });

    _pageController.animateToPage(
      index,
      duration: const Duration(milliseconds: 300),
      curve: Curves.easeInOut,
    );
  }

  @override
  Widget build(BuildContext context) {
    final colorScheme = Theme.of(context).colorScheme;
    final backgroundColor = widget.backgroundColor ?? colorScheme.surface;
    final selectedColor = widget.selectedColor ?? colorScheme.primary;
    final unselectedColor = widget.unselectedColor ?? colorScheme.onSurfaceVariant;

    return Scaffold(
      body: PageView(
        controller: _pageController,
        onPageChanged: (index) {
          setState(() => _currentIndex = index);
        },
        children: _screens,
      ),
      bottomNavigationBar: NavigationBar(
        backgroundColor: backgroundColor,
        elevation: 0,
        selectedIndex: _currentIndex,
        onDestinationSelected: _onTap,
        indicatorColor: selectedColor.withValues(alpha: 0.1),
        destinations: widget.items.asMap().entries.map((entry) {
          final index = entry.key;
          final item = entry.value;
          final isSelected = _currentIndex == index;

          return NavigationDestination(
            icon: Icon(
              item.icon,
              size: widget.iconSize,
              color: unselectedColor,
            ),
            selectedIcon: Icon(
              item.activeIconData,
              size: widget.iconSize,
              color: selectedColor,
            ),
            label: item.label,
          );
        }).toList(),
      ),
    );
  }
}
```

```dart
import 'package:flutter/material.dart';
import 'package:provider/provider.dart';
import '../models/auth_store.dart';

class ProfileScreen extends StatelessWidget {
  const ProfileScreen({super.key});

  @override
  Widget build(BuildContext context) {
    final authStore = Provider.of<AuthStore>(context);
    final user = authStore.user;
    final colorScheme = Theme.of(context).colorScheme;

    return Scaffold(
      appBar: AppBar(
        title: const Text('我的'),
        centerTitle: true,
        elevation: 0,
        backgroundColor: colorScheme.surface,
        actions: [
          IconButton(
            icon: const Icon(Icons.logout),
            onPressed: () async {
              await authStore.logout();
              if (context.mounted) {
                Navigator.pushReplacementNamed(context, '/login');
              }
            },
          ),
        ],
      ),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            CircleAvatar(
              radius: 50,
              backgroundImage: user?.avatar != null ? NetworkImage(user!.avatar!) : null,
              child: user?.avatar == null ? Text(user?.nickname[0] ?? 'U') : null,
            ),
            const SizedBox(height: 20),
            Text(
              user?.nickname ?? '用户',
              style: const TextStyle(fontSize: 24, fontWeight: FontWeight.bold),
            ),
            const SizedBox(height: 10),
            Text(
              user?.phone ?? '未绑定手机',
              style: TextStyle(color: colorScheme.onSurfaceVariant),
            ),
            const SizedBox(height: 20),
            if (user?.isVip ?? false)
              const Chip(
                label: Text('VIP会员'),
                backgroundColor: Colors.amber,
              ),
          ],
        ),
      ),
    );
  }
}
```

```dart
import 'package:flutter/material.dart';
import 'package:lucide_icons/lucide_icons.dart';
import 'package:provider/provider.dart';
import 'widgets/bottom_nav_bar.dart';
import 'models/auth_store.dart';
import 'models/theme_model.dart';
import 'screens/home_screen.dart';
import 'screens/health_screen.dart';
import 'screens/message_screen.dart';
import 'screens/discover_screen.dart';
import 'screens/profile_screen.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MultiProvider(
      providers: [
        ChangeNotifierProvider(create: (_) => ThemeModel()),
        ChangeNotifierProvider(create: (_) => AuthStore()),
      ],
      child: Consumer<AuthStore>(
        builder: (context, authStore, child) {
          return MaterialApp(
            title: '灵伴AI',
            debugShowCheckedModeBanner: false,
            theme: ThemeData(
              useMaterial3: true,
              colorScheme: ColorScheme.fromSeed(
                seedColor: const Color(0xFF2196F3),
              ),
            ),
            home: authStore.isLoggedIn ? const MainScreen() : const LoginScreen(),
            routes: {
              '/login': (context) => const LoginScreen(),
              '/register': (context) => const RegisterScreen(),
            },
          );
        },
      ),
    );
  }
}

/// 主界面（带底部导航栏）
class MainScreen extends StatelessWidget {
  const MainScreen({super.key});

  /// 导航项配置 - 开闭原则：新增项只需在这里添加
  static const List<NavItem> _navItems = [
    NavItem(
      label: '首页',
      icon: LucideIcons.home,
      activeIcon: LucideIcons.home,
      screen: HomeScreen(),
    ),
    NavItem(
      label: '健康',
      icon: LucideIcons.heartPulse,
      activeIcon: LucideIcons.heartPulse,
      screen: HealthScreen(),
    ),
    NavItem(
      label: '消息',
      icon: LucideIcons.messageCircle,
      activeIcon: LucideIcons.messageCircle,
      screen: MessageScreen(),
    ),
    NavItem(
      label: '发现',
      icon: LucideIcons.compass,
      activeIcon: LucideIcons.compass,
      screen: DiscoverScreen(),
    ),
    NavItem(
      label: '我的',
      icon: LucideIcons.user,
      activeIcon: LucideIcons.user,
      screen: ProfileScreen(),
    ),
  ];

  @override
  Widget build(BuildContext context) {
    return BottomNavBar(
      items: _navItems,
    );
  }
}
```

## 开闭原则体现

| 原则 | 实现 |
|------|------|
| **对扩展开放** | 新增导航项只需在 `_navItems` 列表中添加配置 |
| **对修改关闭** | `BottomNavBar` 组件内部代码无需修改 |
| **依赖倒置** | 组件依赖抽象的 `NavItem` 模型，不依赖具体页面 |
| **单一职责** | 导航栏只管切换，页面内容各自独立 |
